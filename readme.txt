WordPress Plugin of Breadcrumbs for Pages,posts,blog.

== Description ==

A wordpress plugin which provide you powerful functionality to integrate breadcrumbs into your site/blog. 
In this plugin you can set display of breadcrumb on homepage by simply pass 1 into function. To display breadcrumb 
on home page use wpeasy_breadcrumb(1), if you dont want to display remove 1 from function.
To butify the breadcrumb add a css class (wpeasybread) into style.css and add your required style.


== Installation ==

You can directly upload the plugin from wordpress admin section or use following steps.
1. Upload plugin folder to the /wp-content/plugins/ directory.
2. Activate the plugin from the 'Plugins' menu in WordPress
3. Place '<?php if(function_exists(wpeasy_breadcrumb)) {wpeasy_breadcrumb(1);} ?>' in your header.php template
